# NCMP – National Constituency Management Platform (Uganda)

## Project Overview

The **National Constituency Management Platform (NCMP – Uganda)** is a national civic digital infrastructure system designed to digitally connect:

- Citizens of Uganda  
- Members of Parliament  
- Government leadership  
- Parliamentary staff  
- Constituency offices  
- National institutions  

into one unified governance and civic engagement platform.

This repository contains the frontend code for the platform built with **React, TypeScript, Vite, TailwindCSS, and ShadCN UI**.

## Local Development

You can work on this project locally using your preferred IDE. Ensure you have **Node.js** and **npm** installed (you can use [nvm](https://github.com/nvm-sh/nvm#installing-and-updating) to manage Node versions).

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start the development server with live reload
npm run dev
